Instrucciones:
1) Copia toda la carpeta en C:\xampp\htdocs\electromisiones_xampp
2) Inicia Apache y MySQL desde XAMPP Control Panel
3) Importa 'electromisiones_db.sql' en http://localhost/phpmyadmin (pestaña Importar)
4) Abre http://localhost/electromisiones_xampp/ para ver la demo.
Notas: usuario MySQL por defecto: root (sin contraseña).
